Overview
This README file details the HTML elements and CSS properties used in the development of a personal portfolio website. It highlights key features like favicon, table, forms, images, and more, demonstrating the webpage's structure and styling.
HTML Elements
Favicon: Represents a shortcut icon, or favicon, of the webpage.
Table: Used for creating tabular data, like the courses section.
Form: Not explicitly present in the provided code, but generally uses <form>, <input>, <label>, <textarea>, <select>, <button>.
Images: Used to embed images in the document.
Hyperlink: Defines hyperlinks to link to other pages or resources.
Button: Not explicitly present but generally represented by <button> or <a> styled with CSS.
Audio/Video: Embeds audio and video files.
Header: Not explicitly defined but can be represented using <header> or a <div> with class .header.
Footer: Defines the footer section of the page.
Summary: Used with <details> to create an interactive widget.
Menu: Can be represented by a list inside a <nav> or a <div>, often used for navigation links.
Tel for Contact Information: Provides a clickable phone link.
Mailto for Contact Information: Provides a clickable email link.
CSS Features
Float and Overflow Property: Used to control layout and handle content overflow. 
Column Layout Structure: Created using classes like .column-layout and .column, showcasing a flexible layout design.
Absolute and Relative Position: for precise positioning of elements.
Responsive Design Using Media Queries: Ensures the website is adaptable to different screen sizes.
