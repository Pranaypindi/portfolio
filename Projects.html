Projects

Waste Material Classification System using CNN:

This project aims to solve the issue of waste material classification by proposing smart frameworks that can efficiently automate the recycling process. The project uses a deep learning algorithm, specifically the ResNet model of Convolutional Neural Network (CNN), to classify waste materials like cardboard, glass, metal, paper, plastic, and trash. The goal is to achieve the best efficiency in garbage processing solutions to ensure appropriate disposal in recyclable and non-recyclable bins.

For more information, check out Waste Material Classification System using CNN

Stock Market Sentiment Analysis:

This project aims to analyze the impact of sentiment on the stock market using Natural Language Processing and Machine Learning techniques. The Top 25 News Headlines are used to extract financial sentiment, and model is trained to evaluate whether the stock price has increased or decreased. The project combines historical stock data and sentiment analysis of news headlines to forecast the future price of a stock of interest. Techniques such as lemmatization and count vectorization are used, and classifiers like Random Forest, Naive Bayes, and Logistic Regression are employed for classification.

For more information, check out Stock Market Sentiment Analysis
